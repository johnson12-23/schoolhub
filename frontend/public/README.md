# Public Assets

Add the hero background image here:

**File:** `hero-bg.jpg`
- Used by LoginPage and RegisterPage as the background hero image
- Recommended dimensions: 1920x1080 or larger
- Recommended size: Optimized JPG under 500KB for fast loading
- The image will be covered/centered on the page

## How to add the image:
1. Save your hero background image as `hero-bg.jpg` in this folder (`frontend/public/`)
2. The LoginPage and RegisterPage will automatically use it as the background
3. A dark overlay (40% opacity) will be applied over the image for text readability
